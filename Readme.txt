HOW TO COMPILE AND RUN Olympic.java

Compile:
	javac -cp "ojdbc7.jar;." Olympic.java
	
Run:
	java -cp "ojdbc7.jar;." Olympic
	

Running the program
	1. To choose a menu choice, type the number next to it and hit enter
	2. You will be prompted to enter the necessary information for each function
	3. Enter the values and hit enter
	4. After each time a function completes, you will be brought back to the 
	   main menu to make another choice
	5. To exit the program, enter 0 on the main menu
